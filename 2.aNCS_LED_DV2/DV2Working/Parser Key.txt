The following is a list of parser commands, in the order they apear in the parsers.
For Strips.
Rst - reset function.
Atm - automode 0 , off. 1 , patterns and palettes. 2 , patterns. 3 , palettes. 4 , pulse mode.
Ats - autoseconds, amount of time between automode calls.
ISt - indivdual strip mode. set to 1 to enable.
SNo - Strip number to address is ISt == 1.
Ptn - pattern number.
Col - our color (RGB).
Pal - palette mode on if 1.
PNo - Number of palette to use.
Ran - will trigger a random pattern.
Pul - Do Pulse.

For Matrix 
Rst - reset function.
Atm - automode 0 , off. 1 , patterns and palettes. 2 , patterns. 3 , palettes. 
Ats - autoseconds, amount of time between automode calls.
Ptn - pattern number.
Col - our color (RGB).
Pal - palette mode on if 1.
PNo - Number of palette to use.
Ran - will trigger a random pattern.
Msg - message for scrolling text.
Mir - mirror mode type. 0 off max 4
